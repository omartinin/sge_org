## Ejercicio 5_7

```python
from functools import reduce

texto = "Hola hola mundo mundo cruel"
texto = texto.lower().split(" ")

def contador(acu, x):
    acu[x] = acu.get(x,0)+1
    return acu

ocurrencias = reduce(contador, texto,{})

print(ocurrencias)
```

## Ejercicio 5_13
```python
from functools import reduce

funciones = [lambda x: x*2, lambda x: x+3, lambda x: x-1]
numeros = [1, 2, 3]
# Resultado esperado: [((1*2)+3)-1, ((2*2)+3)-1, ((3*2)+3)-1] -> [4, 6, 8]
resultado = reduce(lambda acu, funcion: list(map(funcion,acu)),funciones, numeros)
```
